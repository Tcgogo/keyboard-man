> **改键**
- ^ => H
- g_ => L

> **移动**
- shift + l => 移动到行尾 
- shift + h => 移动到行首
- o => 回车向下增加一行
- shift + o => 向上增加一行


> **插入**
- shift + a => 插入到行尾
- shift + i => 插入到行首 


> **操作**
- dd => 删除当前行(相当于 cmd + x)
- yy => 复制当前行
- p => 粘贴
- 与 cmd c ｜ v 不冲突
