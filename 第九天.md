## vim-easymotion
- \<Space>\<Space> + w => 往下进入单词开头跳转
- \<Space>\<Space> + b => 往上进入单词开头跳转
- \<Space>\<Space> + e => 往下进入单词结尾跳转
- \<Space>\<Space> + ge => 往上进入单词结尾跳转
- \<Space>\<Space> + l => 往下进入单词开头和结尾跳转
- \<Space>\<Space> + h => 往上进入单词开头和结尾跳转
- \<Space>\<Space> + j => 全局进入单词开头和结尾跳转

## vim-sneak
- f + 2个字符 => 向下搜索两个字符
- s 改键成 f
- S 改键成 F
- v/normol + f + 2个字符
- opertion + z + 2个字符
