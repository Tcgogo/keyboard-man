## obsidian 双链

### 添加一个内链
- 创建 option + cmd
- 返回 ctrl + o
- [[# 选择标题
- [[| 预览别名
- 预览 cmd + hover
  

### 嵌入
- ![[ 嵌入笔记、音频、视频等

### 基于新的窗格打开链接
- 预览模式 cmd + click
- 编辑模式 cmd + shift + click \ cmd + shift + enter

### 直接打开链接
- option + enter

### note refactor
- cmd + shift + c 把选择的内容添加笔记中
- cmd + shift + n 新建笔记
