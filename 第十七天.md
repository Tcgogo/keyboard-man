## 宏
- 技巧 先规范好光标的位置 移动时使用先相对位置 如webft等
- 开始录制 q + {name} 
- 停止录制 q
- 查看录制好的宏 :reg a
- 触发宏 @ + {name}
- 触发最好一次宏 @@
- 重复执行 数字+@+{name} 
- 报错就停止
- 追加 q 大写的 {name}
- 修改一个宏
  - 取出来 " + {name} + p  / :put + a
  - 修改 " + {name} + yw/yy
  - vscode 不生效，不过也不常用
### 练习

1) Object
2) string
3) number
4) boolean
5) undefined
6) null
