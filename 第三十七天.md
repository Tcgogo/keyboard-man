## chrome devtools

### 打开devtools
- cmd + option + i (记录上次tab的位置)
- cmd + option + j (console.log)
- cmd + shift + c (element 选择查看节点)

### console 面板
- 打开 ctrl + `
- 清空 cmd + k

### 搜索
- 文件 ctrl + p
- 命令 cmd + shift + p

### 切换面板
- cmd + [ or ]

### 在 source panel 中隐藏显示底部面板
- esc
