## 在单文件中移动的更快

### 滚动
- ctrl + f => 向下滚动一屏
- ctrl + b => 向上滚动一屏
- ctrl + d => 向下滚动半屏
- ctrl + u => 向上滚动半屏
- ctrl + e => 向下滚动一行
- ctrl + y => 向上滚动一行
- shift + k => 向上滚动5行(配置)
- shift + j => 向下滚动5行

### 跳转
- zz => 将当前行置于屏幕中间
- zt => 将当前行置于屏幕顶部
- zb => 将当前行置于屏幕底部
- gg => 跳到文件首
- G => 跳到文件尾
- 行数 + g/G => 跳到指定行数
