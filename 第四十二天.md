### normal 模式配置
- 退出 esc / ctrl + [
- 通过环境变量，配置jj jk
- ZVM_VI_INSERT_ESCAPE_BINDKEY
- ZVM_VI_VISUAL_ESCAPE_BINDKEY
- ZVM_VI_OPPEND_ESCAPE_BINDKEY
  
### history
- 查看之前的历史记录 j / k (normal 模式)
- ctrl + p (insert 模式)
- ctrl + n (insert 模式)
- ctrl + r (insert模式)
- 搜索 / 
- n 向前 / N 向后
