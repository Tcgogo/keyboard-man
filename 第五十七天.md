## obsidian 内容编辑 (Markdown快捷键)

### 切换到 read view 模式
- cmd + e
  
### 加粗
- cmd + d

### 斜体
- cmd + i

### link
- cmd + k

### 无序列表
- cmd + - （配置）

### 代码块
- ``
- \``` \```

### 注释
- 选中 + cmd + /

### Todo
- cmd + return
- 移动todo option + ↑/↓

### 编辑title
- f2

### 折叠
- 配置
- zM 全部折叠
- zR 全部展开
- za 单个折叠
