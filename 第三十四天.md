## chrome 搜索与查找

### 显示多功能搜索框 o / O (O会新建标签页打开)
- 配置默认搜索引擎
- 搜索关键字 [key] + space, eg: g 

### 显示搜索框并显示收藏夹的内容 b / B(B会新建标签页打开)

### 显示搜索框并编辑当前网页 ge / gE (gE会新建标签页打开)
- 如果是搜索引擎页面会编辑搜索内容

### 搜索现有标签页 T

### 搜索 / + [value]
- n / N 上下查找

### 在 chrome:/// 页面运行插件
- chrome://flags 页面配置开启 extensions-on-chrome-urls
- Vimium C 勾选权限
